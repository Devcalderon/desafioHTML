Surgimento da Internet

(Voltar)

A História da Internet:

Uma Jornada Fascinante a internet hoje é parte essencial da vida cotidiana, tem uma história rica e surpreendente. Sua origem nos remete à década de 1960, em plena Guerra Fria, com a criação da famosa ARPANET, uma rede militar americana.

A Evolução:

Nas décadas de 1970 e 1980, a ARPANET se expandiu e novos protocolos foram criados, como o TCP/IP, que é base da internet moderna. Surgem os primeiros e-mails, fóruns online e serviços de transferência de arquivos.

A World Wide Web:

Em 1989, Tim Berners-Lee cria a World Wide Web, um sistema de interconexão de documentos de hipertexto. Nascem os navegadores web, como o Mosaic e o Netscape, abrindo as portas para o acesso massivo à internet.

Internet Hoje:

A internet se torna uma rede global, com bilhões de usuários e dispositivos conectados. Surgem novas tecnologias, como banda larga, redes sociais e smartphones, transformando a forma como nos comunicamos, trabalhamos e nos divertimos.

Reflexão:

A história da internet é uma jornada fascinante de inovação e colaboração. Uma rede que começou como um projeto militar se tornou uma ferramenta poderosa para o desenvolvimento social, econômico e cultural.

Como Funciona o HTML

(Voltar)

Imagine o HTML como a alma do seu site, a base que organiza e dá vida a todo o conteúdo que você deseja compartilhar. É como se fosse um esqueleto invisível, moldando a estrutura e definindo o que cada parte da sua página faz.

Blocos de construção:

As tags são os tijolos do HTML, comandos simples que definem cada elemento da sua página. São como etiquetas que informam ao navegador o que cada coisa é: título, parágrafo, imagem, link e muito mais.

Personalização:

Com os atributos, você dá um toque especial aos seus elementos, definindo tamanho, cor, alinhamento e outras características. É como se você estivesse pintando e decorando cada bloco da sua construção. Elementos para todos os gostos: O HTML oferece uma variedade de elementos para você criar páginas completas e atraentes. Títulos, parágrafos, imagens, vídeos, listas, tabelas, links e muito mais são as ferramentas que você tem à mão para dar vida às suas ideias.

Tags

(Voltar)

Tags que aprendi no curso:

Aprendi sobre as tags principais “H1” “H2” servem para criar um cabeçalho ou algum tópico maior, as tags “ol” são utilizadas para criar listas ordenadas e “ul” listas não ordenadas, a tag “li” define um item individual dentro de uma lista, a tag “p” utilizada para iniciar um parágrafo, “a” para links, “blockquote” do HTML é usada para criar uma citação longa retirada de outra fonte, aprendi também a tag “mark” utilizada para destacar textos importantes, sem contar as tags fundamentais como “html” que abre o inicio de todo código html, o head onde fica as configurações do arquivo, e o body onde está todo o corpo do código. As tags HTML são como os blocos de montar de uma página web. Cada bloco tem uma função específica e, quando encaixados, formam a estrutura completa da página,Existem tags para formatação (negrito, itálico), para inserir imagens e muitos outros elementos.

Exemplo:

Bloco "Título": Define o título principal da página, que aparece na guia do navegador. Bloco "Parágrafo": Define um bloco de texto com formatação básica. Bloco "Imagem": Insere uma imagem na página. Bloco "Lista": Cria uma lista numerada ou com marcadores. Assim como os blocos de montar, as tags HTML podem ser combinadas para criar layouts complexos e páginas com diversos elementos.

Resumo:

  • Tags HTML são como instruções para o navegador.
  • Definem a estrutura e o visual da página.
  • São como blocos de montar ou ingredientes de uma receita.
  • A criação de páginas web complexas e com diversos elementos.
